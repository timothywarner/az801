# Module 20: Migrate IIS Workloads to Azure

**Duration:** ~15 minutes

## Learning Objectives

After completing this module, you will be able to:

- Assess IIS workloads by using Azure Migrate
- Migrate IIS workloads to Azure Web Apps
- Migrate IIS workloads to containers
- Deploy Windows containers to Azure Kubernetes Service (AKS)

## Topics Covered

### 20.1. Assess IIS Workloads by Using Azure Migrate
Evaluate IIS applications for Azure readiness.

### 20.2. Migrate IIS Workloads to Azure Web Apps
Move web applications to Azure App Service.

### 20.3. Migrate IIS Workloads to Containers
Containerize IIS applications using Windows containers.

### 20.4. Deploy Windows Containers to Azure Kubernetes Service (AKS)
Run containerized Windows applications on AKS.

## Notes

- Azure App Service provides PaaS hosting for web applications
- Azure Migrate can assess .NET framework compatibility
- Windows containers require Windows Server 2016 or later
- AKS supports both Windows and Linux node pools
- Consider Azure Container Instances for simpler scenarios
