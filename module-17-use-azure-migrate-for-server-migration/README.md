# Module 17: Use Azure Migrate for Server Migration

**Duration:** ~15 minutes

## Learning Objectives

After completing this module, you will be able to:

- Deploy and configure Azure Migrate appliance
- Migrate VM workloads to Azure VMs
- Migrate physical servers to Azure VMs

## Topics Covered

### 17.1. Deploy and Configure Azure Migrate Appliance
Set up the Azure Migrate appliance for discovery and assessment.

### 17.2. Migrate VM Workloads to Azure VMs
Migrate Hyper-V and VMware VMs to Azure.

### 17.3. Migrate Physical Servers to Azure VMs
Perform physical-to-virtual (P2V) migrations to Azure.

## Notes

- Azure Migrate provides a centralized hub for migration
- The appliance performs agentless discovery and assessment
- Agent-based replication is used for actual migration
- Azure Migrate supports dependency analysis for application grouping
- Cost estimates are provided during assessment
