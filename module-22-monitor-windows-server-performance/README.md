# Module 22: Monitor Windows Server Performance

**Duration:** ~20 minutes

## Learning Objectives

After completing this module, you will be able to:

- Monitor Windows Server by using Performance Monitor
- Create and configure Data Collector Sets
- Monitor servers and configure alerts by using Windows Admin Center
- Analyze Windows Server system data by using System Insights
- Manage event logs

## Topics Covered

### 22.1. Monitor Windows Server by Using Performance Monitor
Use Performance Monitor to track system performance counters.

### 22.2. Create and Configure Data Collector Sets
Set up Data Collector Sets to capture performance data over time.

### 22.3. Monitor Servers and Configure Alerts by Using Windows Admin Center
Use Windows Admin Center for real-time monitoring and alerting.

### 22.4. Analyze Windows Server System Data by Using System Insights
Leverage machine learning for predictive analytics on Windows Server.

### 22.5. Manage Event Logs
View, filter, and manage Windows Event Logs for troubleshooting.
